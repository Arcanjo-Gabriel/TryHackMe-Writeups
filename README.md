## Methodology

Every room documented here follows a standard ethical hacking methodology:

1. **Reconnaissance & Enumeration:** Mapping the target and finding entry points.
2. **Exploitation:** Gaining initial access to the system.
3. **Privilege Escalation:** Elevating access to the highest administrative level (Root on Linux / Administrator on Windows).
